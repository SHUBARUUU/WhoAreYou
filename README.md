WhoAreYou is a PHP-based user authentication system built with OOP principles, PDO, and MySQL. Designed as a learning project to understand the fundamentals of backend web developmen, including session management, password security, and page protection.

Themed around Studio Ghibli, each page features a different film as the background:

Spirited Away for login, 
Howl's Moving Castle for register, 
And the duck bath scene for the dashboard.
